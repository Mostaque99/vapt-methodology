# VAPT Methodology & Security Notes

This repository contains my consolidated notes, methodologies, checklists, and templates used for conducting professional Vulnerability Assessments and penetration testing.

## Contents
- **methodology/** – Recon, scanning, enumeration, exploitation, reporting
- **report-template/** – Sanitised report template for client deliverables
- **owasp/** – OWASP Top 10 notes & testing checklist

## Purpose
This repo reflects my ongoing learning in offensive security, secure configuration review, TLS/DNS analysis, and web application testing.

## Tools I Use
Nmap, Nessus, Burp Suite, OWASP ZAP, SSLScan, testssl.sh, Sublist3r, Kali Linux, Ubuntu.
