# OWASP Testing Checklist (Condensed)

## Authentication
- Weak passwords
- 2FA bypass
- Session fixation

## Access Control
- Horizontal privilege escalation
- Vertical privilege escalation
- IDOR testing

## Input Validation
- XSS payloads
- SQLi tests
- Directory traversal attempts

## File Uploads
- MIME spoofing
- SVG/JS payloads
- Malware upload test (simulated)

## Security Headers
- HSTS
- X-Frame-Options
- CSP
- Referrer-Policy

## TLS/SSL
- TLS version check
- Cipher strength
- Certificate validity
