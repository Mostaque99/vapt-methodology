# Enumeration Techniques

## Web Enumeration
- Directory brute force (gobuster/dirsearch)
- Parameter discovery
- File upload module mapping
- Access control testing
- Authentication flows

## DNS Enumeration
- Subdomains
- Records (A, AAAA, MX, TXT, NS)
- Zone transfer testing

## System Enumeration
- OS fingerprinting
- Version mapping to CVEs
- Technology identification (nginx, Apache, Flask, Laravel, etc.)

## Recommended Tools
- dirsearch
- ffuf
- Burp Suite Community/Pro
